# ABYZOR Variations

Generate endless MIDI ideas from your existing MIDI.

ABYZOR Variations is a VST3 MIDI plugin that creates intelligent variations of melodies, chord progressions, basslines and rhythms while preserving the musical identity of your original idea.

Perfect for producers who want to overcome creative blocks, explore new directions and speed up their workflow.

## Features

### Melody Variations

Create alternative melodic phrases while maintaining the original musical character.

### Chord Variations

Generate new harmonic possibilities from existing chord progressions.

### Bassline Variations

Explore different groove and note movement options instantly.

### Rhythm Variations

Transform rhythmic patterns while preserving the feel of the track.

### Intensity Control

Choose how subtle or radical the generated variations should be.

### Genre-Aware Generation

Designed to work across multiple electronic music genres and beyond.

### MIDI Import & Export

Load your MIDI, generate variations and drag the results directly into your DAW.

## Requirements

* Windows 10 or newer
* VST3 compatible host
* DAW with MIDI routing support

Supported DAWs include:

* Ableton Live
* FL Studio
* Cubase
* Studio One
* Bitwig Studio
* Reaper
* Logic Pro (via hosting solutions)

---

## Quick Start

1. Load ABYZOR Variations.
2. Generate, import or drag a MIDI clip.
3. Select a variation type.
4. Adjust Variation Amount and Intensity.
5. Generate.
6. Drag the resulting MIDI into your project.
